Panel Controls Attributes

Data Attribute Description
data-control-type="type" panel control type like 'drop-down', 'textbox', 'placeholder', 'label'
data-control-name="name" panel control name. It should be the same in the top and bottom panels.
data-control-action="action" panel control action like 'sort', 'filter', 'paging'.
data-control-cookies="false" exclude control from cookies (when cookies option is enabled)
